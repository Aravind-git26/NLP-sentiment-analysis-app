# 🎬 IMDB Sentiment Analysis

# Overview
A machine learning–based NLP project that classifies IMDB movie reviews as **Positive** or **Negative** using a Streamlit web application.



# Features
- Text preprocessing (cleaning and tokenization)  
- TF-IDF feature extraction  
- Machine learning model for classification  
- Predicts **Positive / Negative** sentiment  
- Interactive Streamlit web interface  
- Real-time prediction  
- Simple and user-friendly UI
